Frame (Layout) template. Is used from within other templates or even sections. When used it causes the nodes of this template file to become imported to the file being rendered. The including file then contains sections that this Atom renders (precisely like Layouts did before version 3.0). One thing that this does is allow you to render a "main" section from any other file than the template that used this Atom as Layout, decreasing the need to duplicate code if for example you need to render a list+detail combined view as well and still have each view in a separate template, but without the need to move the reusable code to a partial.
