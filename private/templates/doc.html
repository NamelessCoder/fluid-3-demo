Atom/ViewHelper documentation

This view shows a list of every Atom contained in this demo package. Clicking an Atom takes you to a detailed and automatically rendered documentation of the Atom. The indented lists are lists of sections within the parent Atom - these sections are also Atoms which contain the same type of documentation.

Since Fluid 3.0 all templates and ViewHelpers share the same documentation-extracting API.
